# 10 · Estratégia de Monetização

## 1. Modelo

**Freemium + assinatura**, com licenciamento offline-first. Sem anúncios, sem venda de dados (é diferencial de marca).

## 2. Tiers

| | **Free** | **Pro** (assinatura) | **Studio** (assinatura/B2B) |
|---|---|---|---|
| Preço alvo | R$ 0 | ~R$ 24,90/mês ou R$ 199/ano | ~R$ 79,90/mês ou R$ 699/ano |
| Monitoramento tempo real | ✅ básico | ✅ completo (todos sensores) | ✅ completo |
| Diagnóstico / gargalos | ✅ (limitado) | ✅ completo + recomendações | ✅ |
| Otimização | ✅ tweaks `safe` | ✅ catálogo completo + perfis | ✅ |
| Rollback | ✅ | ✅ + histórico estendido | ✅ |
| Benchmark | ✅ 1 suite | ✅ todas + antes/depois | ✅ |
| Game Boost | perfil único | ✅ perfis ilimitados | ✅ |
| Digital Twin / histórico | 7 dias | 1 ano | ilimitado |
| Relatórios | HTML básico | PDF/HTML premium | PDF **white-label** |
| TkAI (Fase 4) | — | ✅ | ✅ |
| Multi-máquina / fleet | — | — | ✅ (Fase 5) |
| Suporte | comunidade | prioritário | dedicado/SLA |

## 3. Justificativa de preço

- **Free** gera adoção e prova de valor (mostra ganho real, mas limita catálogo e histórico).
- **Pro** é o coração: power users e gamers que querem catálogo completo + Digital Twin de 1 ano + Game Boost ilimitado.
- **Studio** mira **assistências técnicas e profissionais de TI** — relatórios white-label para entregar a clientes são o gancho de valor; depois fleet/multi-máquina (Fase 5).

## 4. Gatilhos de conversão

- Após diagnóstico, mostrar quantas otimizações estão "bloqueadas no Free".
- Comparação antes/depois completa só no Pro.
- Relatório bonito para enviar/cliente → Studio.
- Histórico além de 7 dias (Digital Twin) → upgrade natural.

## 5. Distribuição

- Site oficial (download direto, maior margem).
- Microsoft Store (alcance + confiança).
- Programa de afiliados para criadores de conteúdo (gaming/tech).
- B2B direto para assistências (Studio).

## 6. Pagamentos

- Gateways: Stripe (internacional) + provedor local (PIX/boleto/cartão BR).
- Licença = chave assinada; ativação offline-first com verificação online opcional e *grace period* (não trava o usuário se ficar offline).

## 7. Projeção (ilustrativa)

Com 100k downloads → ~3–5% conversão Pro (3–5k assinantes) ≈ R$ 75k–150k/ano recorrente; Studio com ticket maior e churn menor reforça LTV. North-star de monetização: **LTV/CAC > 3** e churn mensal < 4%.

## 8. Anti-padrões evitados (de propósito)

- ❌ Falsos positivos para assustar e vender (reputação CCleaner/SystemCare).
- ❌ Cobrança por "limpar" lixo trivial.
- ❌ Bundling/adware.
- ✅ Valor real, mensurável e honesto = preço sustentável.
